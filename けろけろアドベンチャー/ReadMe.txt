●〇●〇けろけろアクション●〇●〇

マリオのような２Dアクションゲームです。
XInputコントローラ対応。コントローラをUSBに接続していたらコントローラで、接続していなければキーボード操作となります。

各ステージのゴールポイント（旗のポール）に触れたらステージクリアです。
敵は上から踏みつぶすと倒せますが、接触するとゲームオーバーです。ステージ外に落下したり、トゲに触れてもゲームオーバーです。

==============================
●XInputコントローラでの操作方法

ステージ以外での選択画面操作
十字キー（左スティックも同様）：選択
Bボタン：決定

ステージでの操作
十字左右キー（左スティックも同様）：移動
十字上キー（左スティックも同様）：ジャンプ
十字上キー長押し（左スティックも同様）：２段ジャンプ
Xボタン：一時停止

==============================
〇キーボードでの操作方法

ステージ以外での選択画面操作
方向キー：選択
Enter：決定

ステージでの操作
方向キー左右：移動
方向キー上：ジャンプ
方向キー上長押し：２段ジャンプ
Xボタン：一時停止
